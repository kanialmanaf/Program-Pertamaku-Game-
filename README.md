# Program-Pertamaku-Game-
Game sederhana: "Tebak Kosakata"

Deskripsi
- Program menampilkan sebuah kata dari berbagai bahasa.
- Disediakan 5 opsi (nomor 1-5) berisi nama negara.
- Pemain memilih satu opsi yang sesuai dengan negara asal kata tersebut.

Cara menjalankan
1. Jalankan file permainan:

   ```bash
   python tebak_kosakata.py
   ```

2. Ikuti instruksi di layar. Tekan `q` kapan saja untuk keluar.

Catatan
- Game ini berjalan di terminal/command line.
- Bahasa antarmuka menggunakan Bahasa Indonesia.
